== Description ==

This is simple widget of showing youtube subscribe button. Easy to use.

Features include:

* Enable or Disable Full Layout.
* Show or hide subscriber count.



= Installing manually: =

1. Unzip all files to the '/wp-content/plugins/skyboot-youtube-subscribe-button'
2. Log in to WordPress admin and activate the 'skyboot-youtube-subscribe-button' plugin through the 'Plugins' menu
3. Go to widget area and find Skyboot Youtube Subscribe Button and use it.
